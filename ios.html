Set up InSave on iPhone

← Back to InSave

You're set up — InSave is installed. Saving works from the home-screen app.
You're in an in-app browser. Open this page in Safari first — tap the ••• menu → Open in Safari — then continue.

1. Install for reminders

In Safari, tap the Share icon → Add to Home Screen → Add. Open InSave from the new home-screen icon.

2. Turn on reminders

Open InSave from the home screen, then tap Enable reminders and allow notifications.

3. Save reels

Or paste a link (no setup): in Instagram, open a reel → Share → Copy link. In InSave, tap Paste a reel link.
